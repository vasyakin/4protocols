# 4protocols — Универсальный скрипт для Sing-box (4-в-1)

Специализированный скрипт для настройки VPS на базе Sing-box с уникальным набором возможностей:

1. **Гибкая работа с SSL**: Легкое переключение между самоподписанными сертификатами и официальными (Acme).
2. **Туннелирование Cloudflare Argo**: Одновременная поддержка постоянных и временных туннелей для обхода блокировок.
3. **Интеграция Psiphon VPN**: Встроенная возможность перенаправления трафика через Psiphon в 30 различных локаций.

*Основано на базе технологий Hostuno и адаптировано для удобного использования.*
